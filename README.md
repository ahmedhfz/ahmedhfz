## Selected Work

### LumiMate — Agentic AI Productivity Assistant

An AI-powered productivity platform that analyzes tasks, habits, goals,
and user context to provide daily planning, prioritization, memory,
and personalized suggestions.

The project includes authentication, task management, progress tracking,
AI conversations, persistent memory, observability, usage limits,
automated tests, and web/mobile foundations.

`Next.js` `React` `TypeScript` `Expo` `Supabase` `PostgreSQL`
`LLM APIs` `Agentic Workflows` `Sentry` `GitHub Actions`

---

### Solar8 — AI and Cloud Systems

Contributing to production-oriented systems that process IoT-based energy data.

My work includes data pipelines, AWS Lambda services, PostgreSQL,
authenticated REST APIs, energy reporting, frontend integrations,
testing, deployment workflows, and system reliability improvements.

`Python` `FastAPI` `AWS Lambda` `API Gateway` `PostgreSQL`
`React` `TypeScript` `Docker` `GitHub Actions`

---

### NEUROCTULUS — Medical Image Classification

Worked on a CNN-based medical imaging system for detecting stroke
and stroke phases from CT and MRI images.

The project used transfer learning, fine-tuning, preprocessing,
and data augmentation with EfficientNet and ResNet architectures.

**Teknofest 2025 Healthcare AI Finalist — 15th among 37 finalist teams**

`Python` `TensorFlow` `Keras` `CNN` `EfficientNet`
`ResNet` `OpenCV` `Scikit-learn`

---

### CarbonTrack — Carbon Footprint Prediction

Built an end-to-end machine learning application that estimates
individual carbon emissions from user inputs.

The project covered data cleaning, feature preparation, model training,
evaluation, and deployment as an interactive web application.

`Python` `Pandas` `NumPy` `Scikit-learn` `Matplotlib` `Streamlit`
